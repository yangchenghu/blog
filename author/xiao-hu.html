在ubuntu上搭建go环境

搜了一下，发现有一个比较全面的教程

当然还有官方的教程

还是比较喜欢使用apt-get的方式来安装（当然我也是编译了好几次没有成功，T_T）

sudo apt-get install golang

然后添加一下环境，sudo vim /etc/profile

export GOROOT=~/go
export GOARCH=386
export GOOS=linux
export GOBIN=$GOROOT/bin/
export GOTOOLS=$GOROOT/pkg/tool/
export PATH=$PATH:$GOBIN:$GOTOOLS

刷新一下

source /etc/profile

写个Helloword来试试

package main

import "fmt"

func main() {
    fmt.Printf("go print line!\n")
}

保存到test.go，跑一下试试

go run test.go

看这个样子要在home目录下，有go这么个目录，里面还有src文件供运行时查找

好吧

去官方下载

wget https://storage.googleapis.com/golang/go1.5.2.linux-amd64.tar.gz

然后解压到~/go目录下

tar zxvf go1.5.2.linux-amd64.tar.gz ~/go

我们再运行一次呢

go run test.go

ok，成功！

    Rss输出全文

    自从下载了mac下的reeder，烧了FeedHQ，又开始对Rss开始迷恋了

    但发现有些Rss只能输出摘要，这个比较麻烦，还得去再点击一下，去访问网页的内容
